# How to use

```
  converter <the_value> <from_unit> <to_unit>
```
As an example :
```
  converter 1 cm km
```

You can convert mass, temperature and weight units.

Use --list to list all available units

```
Length:
  km (kilometre, kilometres, kilometre, kilometres)
  m (meter, meters, metre, metres)
  cm (centimetre, centimetres, centimetre, centimetres)
  mm (millimetre, millimetres, millimetre, millimetres)
  mi (mile, miles)
  yd (yard, yards)
  ft (foot, feet)
  in (inch, inches)

Temperature:
  C (Celsius, centigrade)
  F (fahrenheit)
  K (kelvin)

Mass:
  kg (kilogram, kilograms)
  g (gram, grams)
  mg (milligram, milligrams)
  lb (pound, pounds)
  oz (ounce, ounces)
  ton (tons, tonne, tonnes, metric ton)
```
